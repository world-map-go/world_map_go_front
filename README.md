# world_map_go_front

## start guide

```
# クローン
git clone git@github.com:world-map-go/world_map_go_front.git

# ディレクトリの移動
cd world_map_go_front

# node_modulesの構築
npm install

# prettierの実行
npm run format

# 起動
npm run dev
```

### コード整形ツール

prettier

```
npm run format
```

ESlint

```
npm run lint
```
